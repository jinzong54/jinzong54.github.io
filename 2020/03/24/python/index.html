对象

  1. Python中一切皆对象
  2. type（obj）获取obj的数据类型
  3. id（obj）获取obj的内存位置
  4. print(obj)打印obj的值
  5. 局部变量优先于全局变量

引用

变量中实际储存的是对象的地址，变量通过地址引用了“对象
变量位于：栈
对象位于：堆”

Python是动态类型的语言
变量不需要显示声明类型。根据变量引用的对象，python解释器自动确定数据类型
Python是强类型语言
每个对象都有数据类型，只支持该类型的操作

删除变量和垃圾回收机制

A=13
del A
A就栈空间被删除，同时堆空间的13被自带的垃圾回收器回收（by某种算法）

字符串

字符串一旦确认就无法改变，通过其他方法产生的是新字符串对象，然后改变原变量的名的索引指向新的字符串。Python不支持char类型,单字符也是字符串
1. 字符串的编码
Python3直接Unicode,可以表示世界任意的字符.python3的字符默认支持的即使16为的Unicode编码,ASCII码是Uicode的子集
ord（“A“）可以转换成Unicode码
2. 成段字符串可以用messge=”””dasdd “asdasd” “””
3. str(5555)>>>>可以将5555转化为字符串
4. 字符串的索引：

  • 正向索引 第一个字符是0 号index。
  • 方向索引 最后一个字符是 -1号 index。
