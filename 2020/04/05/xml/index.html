XML

XML简介：

  • 1、extensible markup language ：可扩展标记型语言：可自定义任何标签
  • 2、xml主要是为了存储传输数据
  • 3、是W3C组织发布的技术
  • 4、xml目前的版本：1.0、1.1（1.0使用较多）

XML的应用：

  • 1、不同系统之间的数据传输
  • 2、用于表示生活中有关系的数据
  • 3、配置文件

XML语法

  • 1、文档声明必须在第一行一列
    • 2、属性
  • version:版本号
  • encoding：编码 utf-8 gbk ISO 8859-1(纯英文编码)
  • 3、中文乱码问题：保存与encoding不一致

xml元素定义

  • 1、基本语法与html保持一致
  • 2、如果标签没有内容可以自结束
  • 3、可以合理嵌套
  • 4、有且只有一个根标签
  • 5、空格，回车，换行也会被解析
  • 6、命名规则：xml区分大小写，html不区分大小写，不能以数字_开头，不能以xml开头，空格冒号

xml中属性的定义

  • 1、参照html属性规范
  • 2、一个标签可以有多个属性
  • 3、属性名称不能相同

xml的注释

  • 1、注释里面不能嵌套注释

xml中的特殊字符

  • 1、< : &lt
  • 2、> : &gt

CDATA 区：解决多个字符都需要转义的操作

PI指令

  • 1、可以引入外部文件
  • 2、<?xml-stylesheet type="text/css" href="1.css"?>
  • 3、使用极少，且不对中文有作用

XML约束

  • 1、约束技术：dtd 和 schema
